# Deen Taqwa — Landing Page

Marketing site for **Deen Taqwa**, an iOS app that gently locks your apps at each of the five daily prayer times. Pray to unlock. Build blessed habits.

**App Store:** https://apps.apple.com/us/app/deen-taqwa-salah-dhikr/id6758683868

## Stack

Static HTML/CSS/JS. No build step. Works out of the box.

- `index.html` — page structure
- `styles.css` — editorial emerald aesthetic
- `script.js` — streak grid + scroll reveals
- `assets/` — app screenshots and preview video

## Design

Editorial-serif direction built around a deep Islamic emerald. Fraunces for display copy, Instrument Sans for body, Amiri for Arabic. Paper grain overlay, asymmetric hero, running marquee of the divine names, a realistic streak visualization, and a full feature walkthrough tying to each app screenshot.

## Run locally

```bash
python3 -m http.server 8000
# open http://localhost:8000
```

## Deploy

Drop the folder on any static host — Vercel, Netlify, GitHub Pages, Cloudflare Pages. No server needed.
